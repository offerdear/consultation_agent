# AI Consultation Agent

How to run this project:

1. **Download as Zip**: Click    [<> Code   ->   Download Zip]
   
2. **Unzip File**: Open with VSCode or other editor
   
3. **Add OpenAI Key**: I will text / email you the OpenAI key. Just copy-paste into the empty **.env** file.
    
4. **Setup Virtual Environment**: Run with whichever python version you have.
   Repeat the last command with [python3 -m pip install ____] until all dependencies are installed. (e.g. python3 -m pip install fitz)
   Make sure you are in the project directory.
```shell
  python3 -m venv venv
  source venv/bin/activate
  python3 -m pip install flask
  ....
```

5. **Run**:
```shell
  python3 app.py
```

6. **Try**: Go to link in the terminal [http://127.0.0.1:5000] or device-specific local development server to try.
